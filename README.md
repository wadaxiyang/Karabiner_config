- Use the `spacebar` as `Fn` key on a Filco Minila keyboard.
    > - `spacebar` + `e` = `up arrow`
    > - `spacebar` + `d` = `down arrow`
    > - `spacebar` + `s` = `left arrow`
    > - `spacebar` + `f` = `right arrow`
    > - `spacebar` + `k` = `home`
    > - `spacebar` + `,` = `end`
    > - `spacebar` + `l` = `page up`
    > - `spacebar` + `.` = `page down`
    > - `spacebar` + `j` = `insert`
    > - `spacebar` + `m` = `delete`
    > - `spacebar` + `;` = `backspace`
